# TalentFlow-AI: Enterprise Recruitment Screening Pipeline

TalentFlow-AI is an intelligent, automated recruitment screening framework built to optimize the initial stages of candidate evaluation. By combining high-performance backend processing with advanced Generative AI, the system parses application data, extracts key professional metrics, and executes automated multi-criteria technical screenings to rank candidates efficiently.

---

## 🚀 Key Features

* **Automated Screening Pipeline:** Asynchronous processing of incoming resume data and candidate profiles.
* **GenAI-Powered Evaluation:** Leverages the **Groq LLaMA** architecture for high-speed, contextual comprehension of technical skill sets.
* **Dynamic Scoring Engine:** Automatically scores and ranks candidates based on customizable enterprise benchmarks (experience, stack alignment, and core competencies).
* **Interactive Recruiter Dashboard:** Clean, intuitive UI built for HR teams to monitor metrics, track candidate pipelines, and review automated screening breakdowns.

---

## 🛠️ Architecture & Tech Stack

* **Backend Framework:** FastAPI (Asynchronous Python Web Gateway)
* **Frontend Interface:** Streamlit 
* **LLM Orchestration:** Groq Cloud SDK (LLaMA Inference Engine)
* **Configuration & Environment Management:** Python-Dotenv & OS-level security layers
